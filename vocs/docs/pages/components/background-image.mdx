## Props

The `BackgroundImage` component extends [`div`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div), so you can pass all standard `div` props.

### `src`

| Description | The src of the background image on the frame                          |
|-------------|------------------------------------------------------------------ |
| Type        | `string` |
| Default     | `undefined`                                                        |

### `width`

| Description | The width of the background image on the frame   |
|-------------|------------------------------ |
| Type        | `string` |
| Default     | `955px`                          |

### `height`

| Description | The height of the background image on the frame   |
|-------------|------------------------------ |
| Type        | `string` |
| Default     | `full`                          |
