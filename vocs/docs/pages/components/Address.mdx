# Address

A simple text component that formats the address, truncating the middle part.

Add `Address` to your project with the following command:

```sh
npx maiton-ui add Address
```

## Props

The `Address` component extends [`Text`](/components/text), so you can pass all the [`TextProps`](/components/text#props).

### `length`

| Description | The number of characters we want to show at the start and end of the address |
| ------------- | ----- |
| Type | `number` |
| Default | `3` |

### `children`

| Description | The address to be formatted |
| ------------- | ----- |
| Type | `React.ReactNode` |
