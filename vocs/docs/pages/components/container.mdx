# Container

<Button href="https://github.com/builders-garden/maiton-ui/blob/main/components/Container.tsx">GitHub</Button>

A flexible container component that can hold other components with customizable padding.

## Installation

```sh
npx maiton-ui add Container
```

## Usage

```tsx
import { Container } from "../components/Container";
```

```tsx
<Container>
  <div tw="flex flex-col align-start gap-4">
    <p>Container</p>
    <p>Your nice container!</p>
  <div>
</Container>
```

## Props

The `Container` component extends [`div`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div), so you can pass all standard `div` props.

### `size`

| Description | The size of the container padding |
|-------------|---------------------------------- |
| Type        | `sm \| md \| lg \| undefined`     |
| Default     | `md`                              |
