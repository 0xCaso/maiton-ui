# Text

Manage a basic paragraph.

Add `Text` to your project with the following command:

```sh
npx maiton-ui add Text
```

## Props

The `Text` component consists of a `p`, so you can pass its default [`props`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/p) props.

### `size`

| Description | The text size |
| ------------- | ----- |
| Type | `sm \| md \| lg` |
| Default | `md` |

### `children`

| Description | The text to be displayed |
| ------------- | ----- |
| Type | `React.ReactNode` |

### `tw`

| Description | The tailwind classes to be applied |
| ------------- | ----- |
| Type | `string` |
